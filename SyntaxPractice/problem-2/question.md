# 2. Separate Strings And Numbers

## Problem Statement

Given an array of tokens containing both strings and numeric values, separate them into two collections.

## Example 1

```text
Input:
tokens = ["John","25","Alice","30","Bob","40"]

Output:
strings = ["John","Alice","Bob"]
numbers = [25,30,40]
```

## Constraints

```text
1 <= tokens.length <= 10^4
```
